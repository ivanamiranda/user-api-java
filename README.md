# 🚀 User API - Spring Boot

API REST simples desenvolvida com Java e Spring Boot para gerenciamento de usuários.

## 🛠️ Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## 📌 Funcionalidades
- Listar usuários
- Criar usuário
- Buscar por ID
- Deletar usuário

## ▶️ Como rodar

1. Execute a aplicação
2. Acesse:
   http://localhost:8080/users

## 📬 Exemplo de requisição

POST /users

{
"nome": "Ivana"
}
